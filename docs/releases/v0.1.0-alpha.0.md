# Codex Sparkompass v0.1.0-alpha.0

Dies ist die erste öffentliche Technical Preview von Codex Sparkompass.

## Was ist Sparkompass?

Sparkompass ist eine lokale Context-Control-Plane für Codex. Das Projekt hilft dabei, vor einem Codex-Lauf weniger unnötigen Kontext zu senden, wichtige Fakten zu schützen, Originalstellen gezielt nachzuladen und Token-Ersparnis mit Qualitätsgates zu verbinden.

## Highlights

- CLI `sparkompass`
- MCP-Server `sparkompass-mcp`
- Lokaler Codex-Plugin-Kandidat mit Skill, MCP-Bridge und Hook-Kandidat
- Verified ContextPacks, Handoff Receipts und PromptPreparation
- TaskOutcome-, ShadowRun-, Evidence-, Ablation- und Slimming-Prüfungen
- Offizielle Codex-JSONL-Usage-Auswertung mit Usage-Invarianten
- Vierarmige Experiment-Matrix: `basis_raw`, `basis_kompakt`, `plugin_raw`, `plugin_kompakt`
- Router für `bypass`, `compact`, `lazy`, `full`
- Tool-Profile: `minimal`, `standard`, `benchmark`, `release`, `debug`
- Gekürzte README mit separatem Nutzungs- und Evidenzhandbuch
- Nüchterne SVG-Diagramme für Struktur, Ablauf, Release-Gates und erste Einsparungen
- Eigenständiges Plugin-`dist`-Bundle für Git-Marketplace-Installationen ohne globale `sparkompass`-CLI
- Git-Marketplace-Smoke mit frischem `CODEX_HOME`, MCP-Lookup und Hook-Redaktionsprüfung
- Redigiertes A/B-Evidence-Bundle unter `evidence/case-studies/readme-ab-v1`
- Release-Audit: `30/30` lokale Anforderungen verifiziert

## Installation

```bash
npm ci
node ./bin/codex-sparkompass.mjs doctor
node ./bin/codex-sparkompass.mjs audit .
```

Vollständige Maintainer-Verifikation:

```bash
npm run check
```

Optional:

```bash
npm link
sparkompass audit .
```

## Einschränkungen

- Alpha-Version, noch nicht stabil.
- Noch nicht offiziell in Codex eingebaut.
- Kein automatisches Umschreiben des internen Codex-Requests.
- `verified-end-to-end-noninferior` ist vorbereitet, aber noch nicht final erreicht.
